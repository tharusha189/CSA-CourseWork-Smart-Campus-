Smart Campus API

The REST API is running successfully.

Base URL: /SmartCampus/api/v1/

Available Endpoints

  • GET /api/v1/ – Discovery & metadata
  • GET /api/v1/rooms – List all rooms
  • POST /api/v1/rooms – Create a room
  • GET /api/v1/rooms/{id} – Get a room
  • DELETE /api/v1/rooms/{id} – Delete a room
  • GET /api/v1/sensors – List sensors (optional ?type=)
  • POST /api/v1/sensors – Register a sensor
  • GET /api/v1/sensors/{id}/readings – Reading history
  • POST /api/v1/sensors/{id}/readings – Add a reading
